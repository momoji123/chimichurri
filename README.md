# Chimichurri
Demo backend support for picanha project.
This project is just a demo for training purpose only.

NOTE: Data like Currency Rate are randomly generated and have no connection what so ever with the real crypto or fiat currency.

# Dev-Preperations
1. Java JDK 20 (Recommended) can be downloaded on: https://jdk.java.net/archive/ or from oracle
2. IDE (Recommended: https://spring.io/tools#eclipse)
3. Clone Project and 
4. import to the IDE (file > import > Maven > Existing Maven Project)
5. Open "Boot Dashboard" tab/view (or if doesnt exist yet: window > show view > other > other > Boot Dashboard)
6. open local > chimichurri and start the server

# Endpoints
1. http://localhost:8080/h2 --> See Database
2. http://localhost:8080/swagger-ui/index.html --> See API Documentation
